Phoebe CMS

Um CMS Headless de Código Aberto em Java & Spring Boot.

Phoebe CMS é um sistema de gestão de conteúdo moderno, API-first, desenhado para criadores de conteúdo e programadores que necessitam de flexibilidade, desempenho e controlo total sobre o frontend.

Funcionalidades Principais

  • API Headless: Uma API REST completa para gestão de conteúdo.
  • Stack Tecnológico Moderno: Construído com Java 21, Spring Boot 3 e Hibernate.
  • Flexível: Entregue conteúdo para qualquer plataforma—web (React, Angular, Vue), mobile, ou IoT.
  • Suporte a Bases de Dados: Funciona com MySQL e PostgreSQL.
  • Código Aberto: Licenciado sob MIT, livre para usar e modificar.
