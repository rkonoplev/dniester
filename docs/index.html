Phoebe CMS

An Open Source Headless CMS built on Java & Spring Boot.

Phoebe CMS is a modern, API-first content management system designed for developers and content creators who need flexibility, performance, and full control over the frontend. Ready to get started? Check out the Quick Start guide.

üîß Key Features

  • Headless API: Complete REST API for content management.
  • Content Management: Full CRUD operations for any content type.
  • Taxonomy System: Categories and tags with flexible filtering.
  • User Management: Role-based access (ADMIN, EDITOR).
  • Security: Spring Security with configurable authentication.
  • Performance: High-performance caching with Caffeine.
  • Rate Limiting: IP-based protection with Bucket4j.
  • Automated DB Migrations: Database schema is managed and versioned with Flyway.
  • Testing: Comprehensive unit and integration test suite.
  • CI/CD: GitHub Actions pipeline with automated testing and code quality checks.
