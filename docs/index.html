Phoebe CMS

An Open Source Headless CMS built on Java & Spring Boot.

Phoebe CMS is a modern, API-first content management system designed for developers and content creators who need flexibility, performance, and full control over the frontend.

Key Features

  • Headless API: A complete REST API for content management.
  • Modern Tech Stack: Built with Java 21, Spring Boot 3, and Hibernate.
  • Flexible: Deliver content to any platform—web (React, Angular, Vue), mobile, or IoT.
  • Database Support: Works with MySQL and PostgreSQL.
  • Open Source: MIT Licensed, free to use and modify.
