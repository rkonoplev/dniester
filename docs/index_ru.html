Phoebe CMS

Headless CMS с открытым исходным кодом на Java & Spring Boot.

Phoebe CMS — это современная, API-first система управления контентом, разработанная для создателей контента и разработчиков, которым нужна гибкость, производительность и полный контроль над фронтендом.

Ключевые особенности

  • Headless API: Полноценный REST API для управления контентом.
  • Современный стек: Создан на Java 21, Spring Boot 3 и Hibernate.
  • Гибкость: Доставка контента на любую платформу — веб (React, Angular, Vue), мобильные приложения или IoT.
  • Поддержка БД: Работает с MySQL и PostgreSQL.
  • Open Source: Лицензия MIT, проект можно свободно использовать и изменять.

Об авторе

Этот проект разработан и поддерживается Романом Коноплёвым. Весь исходный код был написан и проверен лично. Хотя в некоторых частях могли использоваться инструменты ИИ, весь код был верифицирован на соответствие стандартам лицензирования с открытым исходным кодом.
