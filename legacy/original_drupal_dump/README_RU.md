### Описание

В этой папке содержится оригинальный ("сырой") дамп базы данных с сайта на Drupal 6. Этот дамп представляет собой **"точку А"** — исходное состояние данных до начала процесса миграции на новую платформу.

### Назначение

- **Историческая ценность:** Позволяет понять, как выглядели данные в самом начале.
- **Отладка:** Может быть полезен, если в будущем возникнут вопросы о корректности переноса тех или иных данных. Можно сравнить финальные данные с исходными.
